Labels

For our bar and lines chart I felt it was going to be more professional without the color banner next to the label. So I did encounter the config that would help there. 

      
const lineChartOptions = {
    responsive: true,
    plugins: { //The path would be: options.plugins.legend.labels.boxWidth
        legend: {
        labels: {
            boxWidth: 0, //The boxWidth still there but now its width is ZERO! 
                        //as we specified it.
        },
        },
    },
};

Colouring & Styles

As we know at this point, this is quite the powerful library. We can customize our charts easily by some tweaking of our config body or our options body object.

Let's recap on those and see what other options are there! 

      
backgroundColor: [ "rgba(255, 99, 132, 0.2)"] //we change attributes such as backgroundColor

//We can also make use of color strings, those are recognize as well
borderColor: [ "green"]
